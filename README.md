### 📥 Installation Steps

<div style="background: linear-gradient(135deg, #ffecd2 0%, #fcb69f 100%); padding: 2rem; border-radius: 15px; margin: 1rem 0;">
  <div style="display: flex; align-items: center; margin-bottom: 1rem;">
    <div style="background: #ff6b6b; color: white; width: 30px; height: 30px; border-radius: 50%; display: flex; align-items: center; justify-content: center; font-weight: bold; margin-right: 1rem;">1</div>
    <h4 style="margin: 0; color: #2c3e50;">Clone the Repository</h4>
  </div>
  
```bash
git clone https://github.com/aayushvaghela/society-ease.git
cd society-ease
```
</div>

<div style="background: linear-gradient(135deg, #a8edea 0%, #fed6e3 100%); padding: 2rem; border-radius: 15px; margin: 1rem 0;">
  <div style="display: flex; align-items: center; margin-bottom: 1rem;">
    <div style="background: #4ecdc4; color: white; width: 30px; height: 30px; border-radius: 50%; display: flex; align-items: center; justify-content: center; font-weight: bold; margin-right: 1rem;">2</div>
    <h4 style="margin: 0; color: #2c3e50;">Project Structure</h4>
  </div>

```
🏢 society-ease/
├── 🏠 frontend/           # Resident Portal (Next.js 15)
├── ⚙️ admin/             # Admin Portal (Next.js 15)
├── 🔧 backend/           # API Server (Node.js/Express)
├── 📱 mobile/            # Mobile App (React Native) [Coming Soon]
└── 📊 analytics/         # Analytics Service [Coming Soon]
```
</div>

<div style="background: linear-gradient(135deg, #ff9a9e 0%, #fecfef 100%); padding: 2rem; border-radius: 15px; margin: 1rem 0;">
  <div style="display: flex; align-items: center; margin-bottom: 1rem;">
    <div style="background: #45b7d1; color: white; width: 30px; height: 30px; border-radius: 50%; display: flex; align-items: center; justify-content: center; font-weight: bold; margin-right: 1rem;">3</div>
    <h4 style="margin: 0; color: #2c3e50;">Install Dependencies & Setup</h4>
  </div>

```bash
# Install dependencies for all services
npm run install-all

# Set up environment variables
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env.local
cp admin/.env.example admin/.env.local

# Start the development servers
npm run dev
```
</div>
